## 📑 術語對照表(會前發給每位參與者)

| 簡稱 | 白話解釋 |
|------|---------|
| Cortex | 公司內部已運行的 AI 智慧助理平台 |
| 專案類型 | 平台支援多種專案,如「業務報價」「IT 任務」「教育訓練」 |
| Channel | 對話頻道(類 Slack/Teams) |
| 戰情會議室 | 報價案的即時協作畫面 |
| Wizard | 開案精靈,7 步驟 step-by-step 帶 PM 開案 |
| Multi-PM | 多 PM 模型 — 一專案下 DPM/BPM/MPM/EPM 多種 PM |
| DPM/BPM/MPM/EPM | Design / Business / Manufacturing / NPI Engineering PM |
| RACI | Responsible(實作)/ Accountable(背鍋)矩陣 |
| Stage Gate | 階段確認制 |
| Dependency Deadline | 依賴 deadline(例:QA response+1day) |
| RFQ | 客戶詢價單 |
| BOM | 物料清單 |
| NRE | 一次性工程費 |
| Cleansheet | 工廠成本明細表 |
| Tier | 區間帶,如 Tier-A 代表 200K-1M USD |
| 知識庫 / KB | 結案案件存入,未來新案可查 |
| RAG | AI 結合知識庫檢索 |
| SLA | 回覆時效承諾 |
| Status SUMMARY | AI 自動產的專案當下狀態摘要(三處顯示) |
